* CSS 3
* Node.js 12.18.4
* Mailchimp
* Dependencies: {"body-parser": "1.19.0", "express": "4.17.1", and "request": "2.88.2"}

To run this project, install node.js and after install dependencies locally using npm:

```
$ cd ~

~/newsLetterProject $ node "file name.js"
```
